Rust 코드는 함수로 가득하다. 프로그램의 시작점인 `main`도 함수이고, 우리가 직접 정의하는 동작도 대부분 함수로 나눈다.

Rust에서는 함수 이름을 보통 스네이크 케이스(`snake_case`)로 작성한다.

## 함수 정의

함수는 `fn` 키워드로 정의한다.

```rust
fn main() {
    println!("Hello, world!");
    another_function();
}

fn another_function() {
    println!("Another function.");
}
```

Rust는 함수가 소스 코드의 어디에 있느냐보다, 호출 가능한 스코프 안에 정의되어 있는지를 더 중요하게 본다. 즉 `main` 아래에 함수를 정의해도 된다.

## 매개변수

함수는 매개변수 (parameter)를 가질 수 있다. Rust에서는 **매개변수 타입을 반드시 명시**해야 한다.

```rust
fn main() {
    print_labeled_measurement(5, 'h');
}

fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("The measurement is: {value}{unit_label}");
}
```

함수 정의에 쓰인 이름은 매개변수 (parameter)이고, 호출할 때 넘기는 실제 값은 인수 (argument)라고 부른다.

## 구문과 표현식

Rust를 이해할 때 중요한 구분이 있다.

- 구문 (statement): 어떤 동작을 수행하지만 값을 반환하지 않음
- 표현식 (expression): 값을 계산해서 결과를 만듦

예를 들어 `let y = 6;`은 구문이다.

```rust
fn main() {
    let y = 6;
}
```

반면 아래 코드는 표현식이다.

```rust
fn main() {
    let y = {
        let x = 3;
        x + 1
    };

    println!("y = {y}");
}
```

여기서 블록 마지막 줄 `x + 1`은 세미콜론이 없다. 그래서 이 블록의 반환값이 되고, 그 값이 `y`에 바인딩된다.

즉 Rust에서는 블록 자체도 표현식이 될 수 있다.

## 세미콜론의 의미

Rust에서는 세미콜론 하나가 의미를 바꾼다.

```rust
fn main() {
    let x = 5;
}
```

위에서 `let x = 5;`는 구문이다. 값을 반환하지 않는다.

반면 함수나 블록 마지막 줄에서 세미콜론을 빼면 그 줄은 반환값 역할을 한다.

```rust
fn plus_one(x: i32) -> i32 {
    x + 1
}
```

아래처럼 세미콜론을 붙이면 반환값이 아니라 구문이 되어 타입 에러가 난다.

```rust
fn plus_one(x: i32) -> i32 {
    x + 1;
}
```

## 반환값

함수는 값을 반환할 수 있다. 반환 타입은 `->` 뒤에 적는다.

```rust
fn five() -> i32 {
    5
}

fn main() {
    let x = five();
    println!("x = {x}");
}
```

Rust에서는 함수 본문의 마지막 표현식이 암묵적으로 반환된다.

명시적으로 `return`을 써서 일찍 빠져나올 수도 있다.

```rust
fn abs(x: i32) -> i32 {
    if x < 0 {
        return -x;
    }

    x
}
```

하지만 단순한 함수에서는 마지막 표현식 반환 방식이 더 흔하다.

## 함수 호출과 표현식 스타일

Rust는 표현식 중심 언어라서 함수도 "값을 계산하는 단위"로 자연스럽게 쓰인다.

```rust
fn plus_one(x: i32) -> i32 {
    x + 1
}

fn main() {
    let x = plus_one(5);
    println!("x = {x}");
}
```

이 스타일에 익숙해지면 `if`, 블록, 함수가 전부 값을 만들어내는 구조로 읽히기 시작한다.

## 정리

- 함수는 `fn`으로 정의한다.
- 함수 이름은 보통 스네이크 케이스를 쓴다.
- 매개변수 타입은 반드시 명시해야 한다.
- Rust는 구문과 표현식을 구분한다.
- 함수의 마지막 표현식은 반환값이 될 수 있다.
- 마지막 줄 세미콜론 유무가 반환 여부를 바꾼다.
