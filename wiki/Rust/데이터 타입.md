Rust의 모든 값은 특정한 타입을 가진다. Rust는 정적 타입 언어이기 때문에 변수의 타입은 컴파일 시점에 결정되어야 한다. 대부분의 경우 컴파일러가 타입을 추론하지만, 여러 타입이 가능하면 타입 명시가 필요하다.

예를 들어 `parse()`는 결과 타입 후보가 여러 개일 수 있어서 아래처럼 명시해주는 편이 안전하다.

```rust
fn main() {
    let guess: u32 = "42".parse().expect("Not a number!");
}
```

## 타입 분류

Rust의 데이터 타입은 크게 두 가지로 나눌 수 있다.

- 스칼라 타입 (scalar type): 하나의 값만 표현
- 복합 타입 (compound type): 여러 값을 묶어서 표현

## 스칼라 타입

스칼라 타입에는 정수형, 부동 소수점, `bool`, 문자가 있다.

### 정수형

정수형 (integer type)은 소수점이 없는 숫자다.

- 부호 있음: `i8`, `i16`, `i32`, `i64`, `i128`, `isize`
- 부호 없음: `u8`, `u16`, `u32`, `u64`, `u128`, `usize`

특별히 정해진 이유가 없다면 기본값인 `i32`가 보통 무난하다. 인덱스처럼 메모리 크기와 관련된 값에는 `usize`를 자주 쓴다.

```rust
fn main() {
    let x = 10;       // 기본적으로 i32
    let y: u8 = 255;
    let z = 98_222;
}
```

정수 리터럴은 여러 표기법을 지원한다.

```rust
fn main() {
    let decimal = 98_222;
    let hex = 0xff;
    let octal = 0o77;
    let binary = 0b1111_0000;
    let byte = b'A';
}
```

#### 정수 오버플로우

타입 범위를 넘는 값을 다루면 오버플로우가 생긴다.

- 디버그 빌드에서는 보통 패닉이 발생한다.
- 릴리즈 빌드에서는 감싸기(wrapping) 동작이 일어날 수 있다.

Rust는 이 상황을 명시적으로 다루기 위한 메서드도 제공한다.

- `wrapping_*`
- `checked_*`
- `overflowing_*`
- `saturating_*`

### 부동 소수점 타입

Rust의 부동 소수점 타입은 `f32`, `f64` 두 가지다. 기본값은 `f64`다.

```rust
fn main() {
    let x = 2.0;      // f64
    let y: f32 = 3.0;
}
```

### 수치 연산

기본 수치 연산은 일반적인 언어와 비슷하다.

```rust
fn main() {
    let sum = 5 + 10;
    let difference = 95.5 - 4.3;
    let product = 4 * 30;
    let quotient = 56.7 / 32.2;
    let truncated = -5 / 3;
    let remainder = 43 % 5;
}
```

정수 나눗셈은 소수 부분을 버린다.

### bool 타입

Rust의 `bool` 타입은 값이 `true`, `false` 둘뿐이다.

```rust
fn main() {
    let t = true;
    let f: bool = false;
}
```

주로 `if` 같은 조건문에서 사용한다.

### 문자 타입

Rust의 문자 타입은 `char`다. 작은따옴표를 쓰며, 단순 ASCII 문자만이 아니라 유니코드 스칼라 값을 표현한다.

```rust
fn main() {
    let c = 'z';
    let z: char = 'ℤ';
    let heart_eyed_cat = 'A';
}
```

Rust의 `char`는 4바이트이고, 문자열 리터럴과는 다르다.

## compound type

compound type은 여러 값을 하나의 타입으로 묶는다. Rust의 기본 compound type은 튜플 (tuple)과 배열 (array)이다.

### 튜플 타입

튜플은 여러 타입의 값을 하나로 묶을 수 있다. 길이는 고정이다.

```rust
fn main() {
    let tup: (i32, f64, u8) = (500, 6.4, 1);
}
```

패턴 분해로 값을 꺼낼 수 있다.

```rust
fn main() {
    let tup = (500, 6.4, 1);
    let (x, y, z) = tup;

    println!("y = {y}");
}
```

인덱스로 접근할 수도 있다.

```rust
fn main() {
    let tup = (500, 6.4, 1);
    let five_hundred = tup.0;
    let six_point_four = tup.1;
    let one = tup.2;
}
```

값이 전혀 없는 튜플 `()`는 unit 타입이라고 부른다.

### 배열 타입

배열은 모든 요소가 같은 타입이어야 하고 길이도 고정이다.

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];
}
```

타입과 길이를 함께 명시할 수도 있다.

```rust
fn main() {
    let a: [i32; 5] = [1, 2, 3, 4, 5];
}
```

같은 값을 반복하는 배열도 만들 수 있다.

```rust
fn main() {
    let a = [3; 5];
}
```

위 코드는 `[3, 3, 3, 3, 3]`과 같다.

배열 요소는 인덱스로 접근한다.

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];
    let first = a[0];
    let second = a[1];
}
```

배열은 길이가 고정되어 있으므로, 원소 개수가 바뀌지 않는 데이터에 적합하다.

## 정리

- Rust는 정적 타입 언어다.
- 타입은 스칼라 타입과 compound type으로 나뉜다.
- 스칼라 타입에는 정수형, 부동 소수점, `bool`, 문자가 있다.
- compound type에는 튜플과 배열이 있다.
- 배열은 길이가 고정이고, 튜플은 서로 다른 타입을 함께 담을 수 있다.
