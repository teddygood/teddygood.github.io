## [10871 X보다 작은 수](https://www.acmicpc.net/problem/10871)

:::tip
Clicking the heading will take you to the BOJ problem.
:::

## Solution

```cpp
#include <iostream>

using namespace std;

int main()
{
    int n, x;
    cin >> n >> x;

    for (int i = 0; i < n; i++)
    {
        int num;
        cin >> num;
        if (num < x)
        {
            cout << num << " ";
        }
    }
}
```

```python
n, x = map(int, input().split())

integer = list(map(int, input().split()))

for i in range(len(integer)):
    if integer[i] < x:
        print(integer[i], end=" ")
```
