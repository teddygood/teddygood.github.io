## Solution

단순 구현, 수학 문제였다. 알고리즘 다시 시작하기 편한 문제였다.

```python
import math

n = int(input())
size = list(map(int, input().split()))
t, p = map(int, input().split())

count = 0

for s in size:
    count += math.ceil(s / t)

print(count)
print(n // p, n % p)
```
