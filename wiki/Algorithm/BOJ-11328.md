## Solution

문자열은 `sort()`를 사용할 수 없다. `sorted()`를 사용하자.

```python
n = int(input())

for _ in range(n):
    a, b = input().split()
    if sorted(b) == sorted(a):
        print("Possible")
    else:
        print("Impossible")
```
