## Solution

자료구조를 어떻게 정의할지 알면 됐던 문제. 문제를 많이 풀어보지 않아서 h, i를 페어로 정의할 생각을 못 했다. `pair`를 쓰는 것도 정말 오랜만이고 점점 더 익숙해져 보자.

```cpp
#include <iostream>
#include <stack>

using namespace std;

int main() {
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    int n;
    cin >> n;
    stack<pair<int, int>> tower;

    tower.push({100000001, 0});
    for (int i = 1; i <= n; i++) {
        int h;
        cin >> h;
        while (tower.top().first < h) {
            tower.pop();
        }
        cout << tower.top().second << " ";
        tower.push({h, i});
    }
}
```
